# Quality Standards — Blog Content

## Content Quality Scorecard

### 1. Research Quality (25 points)
- [ ] **Source Verification** (10 pts)
  - Minimum 2 sources per major claim
  - Sources are credible and current
  - No hallucinated information
- [ ] **Accuracy** (10 pts)
  - Technical accuracy verified
  - Facts cross-referenced
  - No misleading information
- [ ] **Completeness** (5 pts)
  - Topic thoroughly covered
  - All angles addressed
  - No major gaps

### 2. Writing Quality (25 points)
- [ ] **Clarity** (10 pts)
  - Clear, concise language
  - Active voice preferred
  - No jargon without explanation
- [ ] **Structure** (10 pts)
  - Logical flow
  - Proper headings (H2, H3)
  - Short paragraphs (3-4 sentences)
- [ ] **Grammar** (5 pts)
  - No spelling errors
  - Proper punctuation
  - Correct grammar

### 3. SEO Quality (25 points)
- [ ] **Keyword Optimization** (10 pts)
  - Target keyword in title
  - Keyword in first 100 words
  - Natural keyword density
- [ ] **Meta Information** (10 pts)
  - Optimized title (50-60 chars)
  - Meta description (150-160 chars)
  - URL slug optimized
- [ ] **Linking** (5 pts)
  - Internal links (2-3)
  - External authority links
  - Anchor text optimization

### 4. Engagement Potential (25 points)
- [ ] **Hook** (10 pts)
  - Compelling introduction
  - Clear value proposition
  - Addresses reader pain points
- [ ] **Readability** (10 pts)
  - Grade 8-10 reading level
  - Scannable format
  - Visual breaks (images, code blocks)
- [ ] **Call to Action** (5 pts)
  - Clear next steps
  - Encourages engagement
  - Related content suggestions

## Scoring Guide
- **90-100**: Excellent — Publish immediately
- **80-89**: Good — Minor revisions needed
- **70-79**: Acceptable — Moderate revisions needed
- **60-69**: Needs Work — Major revisions required
- **Below 60**: Poor — Rewrite recommended

## Quality Gates

### Stage 1: Writer Self-Edit
- Remove filler words
- Check sentence structure
- Verify technical accuracy
- Ensure consistent voice

### Stage 2: Editor Review
- Grammar and spelling
- Style consistency
- Structure optimization
- Readability check

### Stage 3: Analyst Review
- SEO optimization
- Quality scoring
- Performance prediction
- Final recommendations

### Stage 4: Orchestrator Approval
- Final quality check
- Brand consistency
- Strategic alignment
- Publish decision

## Common Issues to Check

### Writing Issues
- Filler words (just, really, basically, actually)
- Passive voice overuse
- Long paragraphs
- Jargon without explanation
- Weak introductions

### SEO Issues
- Missing meta description
- Poor keyword placement
- No internal links
- Weak external links
- Over-optimized content

### Technical Issues
- Outdated information
- Incorrect code examples
- Missing prerequisites
- Unclear instructions
- No troubleshooting section

## Improvement Areas

### For Writers
- Study successful blog posts
- Practice concise writing
- Learn SEO basics
- Understand audience needs

### For Editors
- Develop style guide expertise
- Learn readability metrics
- Practice constructive feedback
- Build vocabulary for suggestions

### For Analysts
- Learn SEO tools
- Understand engagement metrics
- Practice data analysis
- Build prediction models
