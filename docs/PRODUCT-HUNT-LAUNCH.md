## 📦 LAUNCH ASSETS

### 1. Tagline (10 words max)
```
Option A: "Blog automation from research to publish with any AI"
Option B: "Automate your blog workflow with any AI provider"
Option C: "Full automation: research → write → SEO → publish"
```

### 2. Description (260 chars)
```
ContentPilot automates your entire blog workflow — from research 
to SEO optimization to publishing. Use any AI provider you want 
(OpenAI, DeepSeek, Ollama, etc). Cross-platform: Blogger + 
WordPress. Free tier available.
```

### 3. Images
- [ ] Logo (240x240px)
- [ ] Gallery images (1270x760px) x 4-5
  - Dashboard screenshot
  - Pipeline flow
  - E-E-A-T scoring
  - Settings page
  - Results/metrics

### 4. Video (optional but recommended)
- [ ] 60-90 second demo video
- [ ] Show: Install → Configure → Run Pipeline → Results
- [ ] Tool: Loom, OBS, or ScreenFlow

### 5. First Comment
```
Hey Product Hunt! 👋

I'm Bagas, a solo developer and blogger.

I built ContentPilot because I was struggling with:
- Spending hours researching topics
- Writing content that doesn't rank
- Managing multiple SEO tools

ContentPilot automates the entire workflow:
1. Research trending topics
2. Write with your AI provider
3. Score with E-E-A-T standards
4. Publish to Blogger/WordPress
5. Monitor Google indexing

Key features:
✅ BYOM — use any AI provider (OpenAI, DeepSeek, Ollama)
✅ Cross-platform — Blogger + WordPress
✅ E-E-A-T scoring — Google-compliant content
✅ Voice guide — maintains your writing style
✅ Free tier available

Would love your feedback! 🙏
```
