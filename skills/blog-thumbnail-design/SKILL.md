# Blog Thumbnail Design (Research-Based)

## Research Basis
- **YouTube**: thumbnails are #1 click factor
- **Mobile**: 60%+ views on small screens
- **Color**: high contrast = higher CTR

## Design Principles

### Text Rules
- **Max 5 words** on thumbnail
- **Large font**: readable at 120x90px
- **Bold/outline**: for visibility
- **Contrast**: text stands out from background

### Color Psychology
- **Red**: urgency, excitement
- **Blue**: trust, professionalism
- **Green**: growth, money
- **Yellow**: attention, optimism
- **Purple**: creativity, luxury

### Composition
- **Rule of thirds**: place key element at intersection
- **Face**: human faces increase CTR
- **Contrast**: thumbnail must stand out in feed
- **Simplicity**: less is more

## YouTube Thumbnails

### Data-Driven Rules
- **Resolution**: 1280x720 minimum
- **Aspect ratio**: 16:9
- **File size**: < 2MB
- **Format**: JPG, PNG, GIF

### Best Practices
- **Face with emotion**: surprise, curiosity, excitement
- **Text overlay**: 3-5 words max
- **Brand consistency**: same colors/fonts
- **A/B test**: test 2-3 variations

### Mobile Rendering
- **Test at 120x90px**: mobile feed size
- **High contrast**: visible on small screens
- **Simple composition**: no tiny details
- **Bold text**: large, readable

## Social Media Thumbnails

### Platform-Specific Sizes
- **Facebook**: 1200x630
- **Twitter**: 1200x675
- **LinkedIn**: 1200x627
- **Pinterest**: 1000x1500

### Design Tips
- **Consistent branding**: same template
- **Platform optimization**: adjust for each
- **Text placement**: avoid platform overlays
- **A/B test**: different platforms, different results

## A/B Testing Thumbnails

### Process
1. Create 2-3 variations
2. Test on small audience
3. Measure CTR difference
4. Roll out winner

### What to Test
- **Text vs no text**
- **Face vs no face**
- **Color variations**
- **Composition changes**

## Tools

### Free
- **Canva**: templates + drag-drop
- **Figma**: design tool
- **GIMP**: image editing

### Paid
- **Adobe Photoshop**: professional
- **Adobe Illustrator**: vector graphics
- **Sketch**: UI design

## Pitfalls
1. Don't use misleading thumbnails (trust damage)
2. Don't use tiny text (unreadable on mobile)
3. Don't ignore platform specifications
4. Don't skip A/B testing
5. Don't use low-quality images

## Verification
- Text readable at 120x90px
- High contrast visible
- Platform-specific sizes used
- A/B testing implemented
- CTR improvement measured
