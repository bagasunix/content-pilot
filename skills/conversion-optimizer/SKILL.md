# Conversion Optimizer (Research-Based)

## Conversion Types for Blogs

### Primary Conversions
- **Newsletter signup**: email subscription
- **Affiliate click**: monetization
- **Download**: lead magnet

### Secondary Conversions
- **Time on page**: > 3 minutes
- **Pages per session**: > 2
- **Return visit**: within 7 days

## CTA Placement Strategy

### Above Fold
- **Position**: after first paragraph
- **Type**: newsletter signup
- **Design**: contrasting color, clear text

### In-Content
- **Position**: after 2nd or 3rd paragraph
- **Type**: contextual CTA (related to content)
- **Design**: button or inline form

### End of Content
- **Position**: after article conclusion
- **Type**: strong CTA (primary conversion)
- **Design**: prominent, clear value proposition

### Sidebar
- **Position**: sticky sidebar
- **Type**: newsletter signup
- **Design**: non-intrusive, visible

## Landing Page Optimization

### Above Fold
- **Headline**: clear value proposition
- **Subheadline**: supporting benefit
- **CTA**: single, clear action
- **Social proof**: if available

### Content Section
- **Benefits**: 3-5 key benefits
- **Features**: supporting details
- **Testimonials**: social proof
- **FAQ**: address objections

### CTA Section
- **Repeat CTA**: multiple opportunities
- **Urgency**: if applicable
- **Risk reversal**: guarantee or free trial

## Funnel Analysis

### Funnel Stages
1. **Awareness**: blog post visit
2. **Interest**: time on page > 2 min
3. **Consideration**: pages per session > 2
4. **Conversion**: newsletter signup or affiliate click
5. **Retention**: return visit within 7 days

### Funnel Metrics
- **Awareness → Interest**: 60% target
- **Interest → Consideration**: 40% target
- **Consideration → Conversion**: 5% target
- **Conversion → Retention**: 30% target

## User Journey Mapping

### Entry Points
- **Organic search**: landing page
- **Social media**: specific post
- **Direct**: homepage
- **Referral**: specific article

### Journey Paths
1. **Single article → Exit** (bounce)
2. **Article → Related article → Exit**
3. **Article → Newsletter signup → Welcome**
4. **Article → Affiliate → Purchase**

### Optimization
- Reduce bounce rate: better hooks
- Increase pages/session: internal links
- Improve conversion: better CTAs
- Increase retention: email nurture

## A/B Testing Framework

### What to Test
- **CTA text**: different copy
- **CTA placement**: above/in/end
- **CTA design**: color, size
- **Page layout**: content structure

### Testing Process
1. **Hypothesis**: "Changing X will improve Y"
2. **Variation**: create alternative
3. **Traffic split**: 50/50 or 80/20
4. **Duration**: 2-4 weeks minimum
5. **Measure**: statistical significance

## Pitfalls
1. Don't test too many things at once
2. Don't ignore mobile conversions
3. Don't forget to track micro-conversions
4. Don't ignore user feedback
5. Don't set and forget — continuous optimization

## Verification
- Conversions tracked in GA4
- CTA placement optimized
- Funnel stages identified
- A/B testing framework established
- Regular optimization reviews
