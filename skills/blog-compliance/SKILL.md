# Blog Compliance (Research-Based)

## Research Sources
- **FTC Guidelines** (Federal Trade Commission)
- **GDPR** (General Data Protection Regulation)
- **Creative Commons** licensing framework
- **DMCA** (Digital Millennium Copyright Act)

## FTC Affiliate Disclosure Requirements

### Rules (Research-Based)
- Must be **clear and conspicuous** near affiliate links
- Must say "I earn a commission" or equivalent
- Can't hide in footer, tiny text, or separate page
- Each post with affiliate links needs disclosure
- Social media: #ad or #sponsored required

### Disclosure Template
```
**Disclosure**: Artikel ini mengandung link afiliasi. Jika kamu membeli melalui link tersebut, saya mendapat komisi tanpa biaya tambahan untuk kamu. Semua rekomendasi berdasarkan pengalaman pribadi.
```

### Placement Rules
- **Blog**: Before first affiliate link
- **Social media**: #ad or #sponsored (visible without expanding)
- **Video**: Verbal disclosure in first 30 seconds
- **Email**: Disclosure near link

## GDPR Compliance (EU Visitors)

### Requirements
1. **Cookie consent banner** — must allow opt-out of non-essential cookies
2. **Privacy policy** — clear data collection explanation
3. **Data deletion requests** — must be honored within 30 days
4. **Data access requests** — provide copy of stored data
5. **Data breach notification** — within 72 hours

### Cookie Categories
- **Essential**: No consent needed (login, security)
- **Analytics**: Consent required (GA4, tracking)
- **Marketing**: Consent required (ads, retargeting)
- **Functional**: Consent required (preferences)

### Implementation
- Use cookie consent manager (CookieYes, CookieBot)
- Block non-essential cookies until consent
- Provide clear opt-out mechanism
- Document consent records

## Image Licensing

### Creative Commons Types
| License | Use | Requirement |
|---------|-----|-------------|
| **CC0** | Any use | None (public domain) |
| **CC-BY** | Any use | Credit author |
| **CC-BY-SA** | Any use | Credit + same license |
| **CC-BY-NC** | Non-commercial | Credit + non-commercial |
| **CC-BY-ND** | No derivatives | Credit + no modification |

### Fair Use (Limited)
- Transformative use (criticism, commentary)
- Non-commercial educational use
- Small portion of work
- **Risk**: High for commercial blogs — avoid if possible

### AI-Generated Images
- Current legal gray area
- Check platform terms (DALL-E, Midjourney, Stable Diffusion)
- Commercial use may have restrictions
- Document generation parameters for transparency

## DMCA Procedures

### Receiving Takedown Notice
1. Review claim validity
2. Remove content if valid (within 24 hours)
3. Notify original author
4. Counter-notice if claim is false

### Sending Takedown Notice
1. Identify infringing content
2. Document original work (screenshots, timestamps)
3. Send DMCA notice to hosting provider
4. Follow up if no response in 10 business days

### Prevention
- Create original content
- Use properly licensed images
- Document all sources
- Maintain content originality records

## Content Originality

### Plagiarism Prevention
- Write from own knowledge and experience
- Cite all sources with links
- Use plagiarism checker before publish
- Document research sources

## Pitfalls
1. **FTC**: Disclosure must be near link, not in footer
2. **GDPR**: Cookie consent must be opt-in, not pre-checked
3. **Images**: CC-BY-NC = no commercial use (AdSense = commercial)
4. **DMCA**: Respond within 24 hours or risk hosting takedown
5. **AI content**: No clear legal framework yet — document everything

## Verification
- Affiliate disclosure present near every link
- Cookie consent banner blocks non-essential until opt-in
- All images properly licensed and attributed
- Privacy policy updated and accessible
- Content originality verified
