# Blog Brand Visual (Research-Based)

## Research Basis
- **Consistency**: 80% higher brand recognition
- **Trust**: professional visuals = credibility
- **Memory**: visual identity aids recall

## Brand Elements

### Color Palette
- **Primary**: 1-2 main colors
- **Secondary**: 2-3 supporting colors
- **Neutral**: 1-2 background/text colors
- **Accent**: 1 highlight color

### Typography
- **Heading**: 1 font family (bold, distinctive)
- **Body**: 1 font family (readable, clean)
- **Accent**: optional decorative font
- **Scale**: consistent size hierarchy

### Logo Usage
- **Primary**: full logo with tagline
- **Secondary**: icon only
- **Minimum size**: 120px width
- **Clear space**: logo breathing room

## Style Guide Creation

### Color Specifications
```
Primary: #1E3A8A (blue-900)
Secondary: #3B82F6 (blue-500)
Accent: #F59E0B (amber-500)
Background: #FFFFFF
Text: #1F2937 (gray-800)
```

### Typography Scale
```
H1: 2.5rem / 40px
H2: 2rem / 32px
H3: 1.5rem / 24px
Body: 1rem / 16px
Small: 0.875rem / 14px
```

### Spacing System
```
xs: 4px
sm: 8px
md: 16px
lg: 24px
xl: 32px
2xl: 48px
```

## Visual Consistency

### Image Style
- **Photography**: bright, clean, professional
- **Illustrations**: flat, modern, consistent
- **Icons**: consistent style (line, filled, etc.)
- **Graphics**: brand colors, consistent layout

### Template System
- **Blog header**: consistent per category
- **Social media**: branded templates
- **Newsletter**: branded email template
- **Infographics**: branded data visualization

## Brand Application

### Website
- **Header**: logo + navigation
- **Footer**: brand colors + links
- **Content**: consistent typography
- **Images**: consistent style

### Social Media
- **Profile picture**: logo icon
- **Cover photo**: brand colors + tagline
- **Posts**: branded templates
- **Stories**: consistent style

### Marketing Materials
- **Business cards**: brand colors + logo
- **Presentations**: branded template
- **PDFs**: branded document template
- **Email**: branded newsletter template

## Pitfalls
1. Don't use too many colors (3-5 max)
2. Don't mix too many fonts (2-3 max)
3. Don't ignore accessibility (contrast ratios)
4. Don't forget mobile rendering
5. Don't skip style guide documentation

## Verification
- Color palette defined and documented
- Typography scale consistent
- Logo usage guidelines clear
- Templates created for common uses
- Style guide accessible to team
