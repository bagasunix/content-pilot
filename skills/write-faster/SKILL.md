# Write Faster — Practical Speed Techniques

## The 10 Techniques

### 1. Use TK Placeholders (Journalist Trick)
- Write from head, drop "TK" wherever stuck on facts
- "TK" = "to come" — rare letter combo in English
- Draft first, fill holes later
- **Why it works:** Prevents over-researching, keeps flow going

### 2. Turn Off Spellcheck/Grammar Check
- Squiggly red lines yank you out of flow
- Grammar checker screams at intentional choices
- Turn OFF during drafting, check AFTER
- **Why it works:** Eliminates distraction, maintains momentum

### 3. Use Text Expansion
- Auto-expand shortcuts into full text
- "bbio" → full bio (4 keystrokes vs 80)
- Use double-letter prefix to avoid conflicts
- **Why it works:** Fewer keystrokes = faster writing

### 4. Do B-Minus Work
- Don't aim for perfect first draft
- Get it DOWN, then make it BETTER
- Editing is easier than creating
- **Why it works:** Perfectionism kills speed

### 5. Write Under Pressure
- Set timer (Pomodoro: 25 min)
- Gamble with reputation ("I'll publish this")
- Commit to deadline
- **Why it works:** Pressure forces focus

### 6. Skip Bathroom Breaks (Seriously)
- Don't stop until draft is done
- Physical urgency = writing speed
- (Use at own risk!)
- **Why it works:** Eliminates interruptions

### 7. Get Zen Before Writing
- Clear mind before sitting down
- 5 min meditation or walk
- Remove phone from room
- **Why it works:** Focused mind = faster output

### 8. Stop Letting Schedule Push You Around
- Block writing time on calendar
- Protect it like a meeting
- Write FIRST, do other things after
- **Why it works:** Priority = production

### 9. Play Word Games
- Type racing games improve WPM
- Practice typing speed
- **Why it works:** Faster typing = faster writing

### 10. Gamble with Your Reputation
- Publish before you're ready
- Commit publicly to deadlines
- Social pressure = motivation
- **Why it works:** External accountability

## Quick Wins (Do Today)
- [ ] Turn off grammar checker during drafting
- [ ] Set up 3 text expansion shortcuts (bio, sign-off, common phrase)
- [ ] Use TK placeholders instead of stopping to research
- [ ] Set 25-minute timer and write without stopping

## For AI-Assisted Writing
- Generate draft with AI → then edit with these techniques
- AI handles first draft (B-minus work), you refine
- Use TK for AI gaps → fill manually
- Speed comes from EDITING, not generating
