# Blog Production Pipeline

## Pipeline Architecture

### 7-Stage Pipeline
```
Idea → Research → Brief → Draft → Review → Visual → Publish
```

### Stage Details

**Stage 1: Idea (30 min)**
- Input: trending topics, analytics signals
- Process: generate 5-10 ideas, score by value
- Output: ranked idea list

**Stage 2: Research (2 hours)**
- Input: ranked idea
- Process: keyword research, competitor analysis
- Output: research brief

**Stage 3: Brief (30 min)**
- Input: research findings
- Process: content outline, SEO plan
- Output: content brief

**Stage 4: Draft (4 hours)**
- Input: content brief
- Process: write from brief, apply voice
- Output: first draft

**Stage 5: Review (2 hours)**
- Input: first draft
- Process: fact check, SEO review, edit
- Output: reviewed draft
- See `references/blog-editor.md` for full review workflow

**Stage 6: Visual (1 hour)**
- Input: reviewed draft
- Process: images, thumbnails
- Output: visual assets

**Stage 7: Publish (30 min)**
- Input: draft + visuals
- Process: upload, format, schedule
- Output: live article

## Content Types

### Evergreen Content
- **Purpose**: long-term traffic
- **Examples**: tutorials, guides, how-tos
- **Update**: quarterly refresh

### Trending Content
- **Purpose**: immediate traffic
- **Examples**: news, updates, releases
- **Update**: real-time

### Engagement Content
- **Purpose**: community building
- **Examples**: opinions, stories, discussions
- **Update**: as needed

### Refresh Content
- **Purpose**: maintain existing traffic
- **Examples**: updated old articles
- **Update**: based on decay signals

## Quality Gates

### Gate 1: Idea → Research
- [ ] Topic has search demand
- [ ] Business potential > 1
- [ ] Not duplicate with existing content

### Gate 2: Research → Brief
- [ ] Keywords validated
- [ ] Competitor gaps identified
- [ ] Unique angle clear

### Gate 3: Brief → Draft
- [ ] Content outline complete
- [ ] SEO plan included
- [ ] Internal links identified

### Gate 4: Draft → Review
- [ ] Word count 1000-2000
- [ ] All sections covered
- [ ] Voice guide applied

### Gate 5: Review → Visual
- [ ] Fact check passed
- [ ] SEO optimization complete
- [ ] Readability score > 60

### Gate 6: Visual → Publish
- [ ] Images optimized
- [ ] Alt text included
- [ ] Schema markup ready

### Gate 7: Publish → Promote
- [ ] URL submitted to GSC
- [ ] Social media scheduled
- [ ] Newsletter included

## Pipeline Metrics

### Per Article
- **Total time**: target < 24 hours
- **Stage time**: track each stage
- **Revision count**: target < 2
- **Quality score**: 1-10 rating

### Aggregate
- **Articles per week**: target 2-3
- **On-time rate**: > 90%
- **Quality trend**: improving
- **Traffic impact**: per article

## Pitfalls
1. Don't skip stages (quality suffers)
2. Don't rush research (poor foundation)
3. Don't ignore quality gates
4. Don't forget to promote after publish
5. Don't optimize speed over quality
