# Content Brief (Research-Based)

## Research Basis
- **Backlinko**: "The more direction you provide upfront, the less time you'll spend on revisions"
- **HubSpot**: content briefs reduce revision cycles by 50%
- **Industry**: comprehensive briefs = higher first-draft acceptance rate

## Why Content Briefs Matter
- Eliminate guesswork for writers
- Align entire team on purpose and audience
- Prevent scope creep
- Ensure every piece serves a business goal
- Reduce revision cycles

## Brief Components

### 1. Topic & Purpose
```
Topic: [Specific topic]
Primary Goal: [Traffic / Leads / Conversions / Brand Awareness]
Business Connection: How this content ties to revenue
Target Awareness Stage: [Most / Product / Solution / Problem / Unaware]
```

### 2. Audience Insights
```
Target Audience: [Who specifically]
Expertise Level: [Beginner / Intermediate / Advanced]
Pain Points:
  1. [Top pain point]
  2. [Second pain point]
  3. [Third pain point]
Questions They're Asking:
  - "Question from Reddit/Quora/support tickets"
  - "Question from search autocomplete"
  - "Question from PAA"
```

### 3. SEO Parameters
```
Primary Keyword: [keyword]
Search Volume: [number]
Keyword Difficulty: [score]
Search Intent: [Informational / Commercial / Transactional]
Secondary Keywords:
  1. [keyword 1] — [volume]
  2. [keyword 2] — [volume]
  3. [keyword 3] — [volume]
Target Word Count: [range]
```

### 4. SERP Analysis
```
Current Top 5 Results:
1. [URL] — [word count] — [what they cover well]
2. [URL] — [word count] — [what they cover well]
3. [URL] — [word count] — [what they cover well]
4. [URL] — [word count] — [what they cover well]
5. [URL] — [word count] — [what they cover well]

Gaps to Fill:
- [Topic/angle competitors miss]
- [Data point not covered]
- [Audience segment not addressed]

SERP Features to Target:
- [ ] Featured snippet (paragraph/list/table)
- [ ] People Also Ask (collect 10+ questions)
- [ ] Image pack
- [ ] Video carousel
```

### 5. Information Gain Plan
**What unique value will THIS article provide?**

```
Expert Quotes:
  - [Expert name], [title] — topic to ask about
  - [Expert name], [title] — topic to ask about

Original Data:
  - [Survey/study/data point to create or find]
  - [Case study from own experience]

Unique Angles:
  - [Angle competitors haven't covered]
  - [Fresh perspective or contrarian view]

Interactive Elements:
  - [ ] Template/checklist
  - [ ] Calculator/tool
  - [ ] Infographic/chart
```

### 6. Content Structure
```
H1: [Title with primary keyword]
  H2: [Section 1 — address top pain point]
    H3: [Subsection]
    H3: [Subsection]
  H2: [Section 2 — solution/approach]
    H3: [Subsection]
  H2: [Section 3 — examples/case study]
  H2: [Section 4 — advanced tips]
  H2: [FAQ section — use PAA questions]
  H2: [Conclusion with CTA]

Estimated Length: [word count]
Internal Links to Include:
  - [existing article 1] — anchor text
  - [existing article 2] — anchor text
External Sources to Cite:
  - [source 1]
  - [source 2]
```

### 7. E-E-A-T Requirements
```
Experience: [How to demonstrate first-hand experience]
Expertise: [What credentials/sources to include]
Authoritativeness: [What builds authority in this piece]
Trustworthiness: [What signals trust — data, citations, transparency]
Author Bio: [Relevant experience to highlight]
```

### 8. Repurposing Plan
```
Social Media:
  - Twitter/X thread: [5-10 key points]
  - LinkedIn post: [professional angle]
  - Instagram carousel: [visual summary]

Newsletter:
  - Excerpt: [best 2-3 paragraphs]
  - CTA: [what to promote]

Other:
  - [ ] Infographic from data
  - [ ] Video script from key points
  - [ ] Quora answers linking to article
```

## Brief Quality Checklist
- [ ] Primary keyword has search demand
- [ ] Audience pain points identified from real data
- [ ] SERP analyzed (top 5+ competitors reviewed)
- [ ] Information gain plan includes unique value
- [ ] Content structure matches search intent
- [ ] E-E-A-T requirements defined
- [ ] Repurposing plans included
- [ ] Internal links identified
- [ ] External sources cited

## Workflow
1. Research: keyword + SERP + audience questions
2. Analyze: competitor content, gaps, opportunities
3. Draft brief: all 8 components above
4. Review: ensure brief is comprehensive enough
5. Hand off to writer: with all context included
6. Writer produces: first draft matching brief
7. Review: compare draft to brief requirements

## Pitfalls
1. Don't skip audience research — write for real people
2. Don't ignore SERP analysis — Google shows what works
3. Don't forget information gain — what makes THIS article unique
4. Don't skip repurposing plan — maximize content investment
5. Don't make briefs too rigid — leave room for writer creativity

## Verification
- All 8 components completed
- Brief reviewed before writer handoff
- Writer has enough context to produce first draft
- Information gain plan includes at least 2 unique elements
