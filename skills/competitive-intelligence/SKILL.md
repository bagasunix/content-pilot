# Competitive Intelligence (Research-Based)

## Research Basis
- **Backlinko 2026**: "Run a cross-platform competitive analysis — spy on competitors platform by platform"
- **Ahrefs**: competitor keyword loss harvesting = quick wins
- **Industry**: systematic CI = 2-3x faster content strategy

## What Is Competitive Intelligence

### Definition
Systematic collection and analysis of competitor data across all platforms to identify opportunities, threats, and strategic advantages.

### Why It Matters
- **Skip guesswork**: competitors already validated what works
- **Find gaps**: what they miss = your opportunity
- **Speed**: leverage their research investment
- **Defense**: know what competitors are planning

## Cross-Platform Analysis

### Platform Checklist
| Platform | What to Analyze | Tools |
|----------|----------------|-------|
| Blog | Top pages, topics, content format | Ahrefs, SEMrush |
| YouTube | Popular videos, topics, engagement | TubeBuddy, VidIQ |
| LinkedIn | Posts with high engagement | Manual scan |
| Podcasts | Episode topics, guest names | Apple Podcasts, Spotify |
| Social Media | Post performance, hashtags | BuzzSumo, manual |
| Backlinks | Link sources, anchor text | Ahrefs, Majestic |

### Step-by-Step Per Platform

**Blog Analysis**
```
1. Ahrefs → Site Explorer → competitor domain
2. Organic Pages → sort by traffic
3. Note: top 20 pages, keywords, traffic
4. Content format: guide, listicle, tutorial, review
5. Word count range of top content
6. Update frequency
```

**YouTube Analysis**
```
1. Go to competitor YouTube channel
2. Sort by "Popular" — see top videos
3. Note: topics, video length, thumbnails
4. Check: view count, engagement (likes/comments)
5. Note: upload frequency, series format
```

**LinkedIn Analysis**
```
1. Find competitor company page
2. Click "Posts" — sort by engagement
3. Note: post format, topic, frequency
4. Check: comment quality, repost count
5. Note: what gets engagement vs what doesn't
```

**Podcast Analysis**
```
1. Search Apple Podcasts / Spotify for competitor
2. Note: episode topics, guest names
3. Check: episode length, frequency
4. Note: most popular episodes (by reviews/plays)
```

## Keyword Loss Harvesting

### What It Is
Find keywords your competitors USED to rank for but lost. Create better content and take those rankings.

### Process
```
1. Ahrefs → Site Explorer → competitor domain
2. Organic Keywords → Position Changes
3. Filter: "Lost" and "Declined"
4. Review: keywords they stopped ranking for
5. For each keyword:
   a. Check current top 10
   b. Assess: can you create better content?
   c. Estimate: traffic potential
   d. Prioritize: high volume + low difficulty
6. Create content targeting lost keywords
```

### Why It Works
- Competitor already validated search demand
- Their content became outdated or was removed
- Lower competition (they gave up on it)
- Proven topic = faster results

## Content Format Analysis

### Identify Winning Formats
For each competitor, note:
- **What format** gets most traffic? (guide, listicle, comparison, tutorial)
- **What length** performs best? (1000, 2000, 3000+ words)
- **What structure** ranks? (H2/H3 patterns, lists, tables)
- **What visual style** works? (screenshots, infographics, videos)

### Format Benchmarking Template
```
Competitor: [domain]
Top Format: [guide/listicle/comparison]
Avg Word Count: [number]
Avg Backlinks: [number]
Content Freshness: [how often updated]
Visual Style: [screenshots/infographics/videos]
CTA Pattern: [what they promote]
```

## Backlink Profile Comparison

### What to Analyze
1. **Total backlinks**: how many links do they have?
2. **Referring domains**: how many unique sites link to them?
3. **Link sources**: where do their links come from?
4. **Anchor text**: what keywords do links use?
5. **Link velocity**: how fast are they gaining/losing links?

### Gap Analysis
```
Their backlinks - Your backlinks = Link gap
Link gap = Outreach opportunities

Find sites linking to competitors but not you:
1. Ahrefs → enter competitor → Backlinks
2. Export top 100 backlinks
3. Filter: sites not linking to you
4. Outreach: offer better content or resource
```

## Workflow

### Monthly Competitive Review
1. **Data collection**: update competitor metrics
2. **Keyword harvesting**: find lost keyword opportunities
3. **Content gap analysis**: identify missing topics
4. **Backlink gap analysis**: find outreach targets
5. **Platform monitoring**: track competitor activity
6. **Report generation**: compile findings
7. **Action items**: prioritize and assign

## Pitfalls
1. Don't copy competitors — improve and differentiate
2. Don't analyze only one platform — cross-platform is key
3. Don't forget keyword loss harvesting — easiest wins
4. Don't ignore small competitors — they often have good ideas
5. Don't skip monthly updates — competitive landscape changes

## Verification
- All top competitors analyzed across platforms
- Keyword loss opportunities identified
- Content gaps documented
- Backlink opportunities listed
- Monthly report generated
- Action items prioritized and assigned
