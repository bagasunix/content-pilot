# Hook Mastery (Research-Backed)

## Core Framework: AIDA Model
**Attention** → **Interest** → **Desire** → **Action**

Every piece of content follows this flow:
1. **Attention**: Hook that stops the scroll
2. **Interest**: Promise of value
3. **Desire**: Emotional connection
4. **Action**: Clear next step

## Headline Formulas (BuzzFeed/Upworthy Data)

### Formula 1: [Number] + [Adjective] + [Noun] + [Promise]
- "7 Cara Mudah Meningkatkan Traffic Blog"
- "5 Tools SEO Gratis yang Jarang Diketahui"
- "3 Kesalahan Fatal yang Membunuh Blog Kamu"
**Why it works**: Numbers create specificity, adjectives add emotion

### Formula 2: How to [Achieve] Without [Pain Point]
- "Cara Mendapatkan Backlink Tanpa Spam"
- "Cara Menulis Artikel 2000 Kata Tanpa Writer's Block"
**Why it works**: Addresses fear while promising solution

### Formula 3: [Year] Guide: [Topic] for [Audience]
- "Panduan SEO 2026 untuk Blogger Pemula"
- "Tutorial WordPress 2026 Lengkap untuk Pemula"
**Why it works**: Freshness signal + audience targeting

### Formula 4: [Topic]: [Number] Things You're Doing Wrong]
- "SEO: 5 Kesalahan yang Membunuh Traffic"
- "Blog: 7 Alasan Kenapa Visitor Kamu 0"
**Why it works**: Fear of missing out + self-improvement

## Curiosity Gap Technique

### Structure
**State something unexpected** → **Don't reveal answer** → **Promise revelation**

### Examples
- "90% Blog Tidak Pernah Dapat Traffic — Ini Alasannya"
- "Saya Kehilangan 50% Traffic Semalam — Ternyata Ini Penyebabnya"
- "Rahasia Blogger Sukses yang Tidak Pernah Dibahas"

### Rules
- Must deliver on promise in content
- Don't be misleading (clickbait without substance)
- Specificity > vagueness ("90%" > "banyak")

## Pattern Interrupt Technique

### Structure
**Break expected pattern** → **Surprising statement** → **Explain**

### Examples
- "SEO Bukan Soal Backlink. Ini Yang Sebenarnya."
- "Stop Menulis Artikel Setiap Hari. Ini Kenapa."
- "Saya Tidak Pernah Pakai SEO Tools. Traffic Naik 300%."

## First Line Hooks (Article Opening)

### Type 1: Bold Statement
"Saya kehilangan semua traffic dalam semalam."
**Impact**: Creates immediate concern/curiosity

### Type 2: Question
"Pernah merasa blogmu tidak ada yang baca?"
**Impact**: Personal connection, reader nods

### Type 3: Data Point
"Rata-rata blogger menulis 15 jam per artikel. Saya menulis dalam 2 jam."
**Impact**: Specificity creates credibility

### Type 4: Story
"Dulu saya pikir SEO itu rumit. Ternyata..."
**Impact**: Relatable, creates journey narrative

## Platform-Specific Hooks

### Article Title
- Max 60 characters for SEO
- Include target keyword (front-loaded)
- Emotional trigger + specificity

### Video (First 10 Seconds)
- "Jangan [common mistake] sebelum nonton ini"
- "[Shocking statement]. Ini alasannya..."
- "Dalam [time], saya akan tunjukkan [benefit]"

### Thumbnail (Max 5 Words)
- Large readable font
- High contrast colors
- Numbers or power words
- Test at 120x90px (mobile feed size)

### Social Media
- **Twitter**: Question or stat hook
- **LinkedIn**: Professional insight
- **WhatsApp**: Emoji + short hook + link

## Pitfalls
1. Don't clickbait without delivering — match content to hook
2. Test multiple hooks — don't settle for first draft
3. Avoid generic hooks — be specific
4. Cultural context — hooks work differently per audience
5. A/B test when possible — track CTR in GSC

## Verification
- Hook makes reader want to click/read
- Content delivers on hook's promise
- Platform-appropriate format
- Keywords included where needed
